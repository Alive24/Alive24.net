# Alive24.net

本站立项于2023-01-07，使用`Vue 3` + `Nuxt 3` + `DaisyUI` + `TailwindCSS`开发，作为我的个人网站，用于支持我的自由职业者生涯发展（包括展示业务、更新动态），以及发布我的写作内容。

## 运维 DevOps
- 本站为全静态。为了未来可能接入国内公众号提前进行备案，并进行了境内境外分流。
- 境外站点使用Vercel直接根据commit构建，境内计划使用`Jenkins` + `Docker`自动部署，或使用OSS。
